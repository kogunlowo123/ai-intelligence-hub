## Deployment

This is a single `index.html` file. Deploy anywhere:

- **GitHub Pages**: Push to `main` branch, enable Pages in repo settings
- **Netlify/Vercel**: Drag and drop the file
- **Local**: Open `index.html` in any browser

## Tech Stack

- Vanilla HTML/CSS/JS — no frameworks
- Google Fonts (Outfit + JetBrains Mono)
- CSS Grid + Flexbox layout
- CSS custom properties for theming
- Intersection-based animations
